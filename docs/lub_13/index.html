Цель работы

Освоить базовые возможности библиотеки NumPy для работы с многомерными массивами, выполнить векторные и матричные операции, провести статистический анализ данных и визуализировать результаты.

Задачи

  • Создание массивов (одномерных, двумерных) с помощью np.arange, np.random.rand, изменение формы и транспонирование.
  • Реализация векторных операций: сложение, умножение на скаляр, поэлементное умножение, скалярное произведение.
  • Матричные операции: умножение матриц, вычисление определителя, обратной матрицы, решение системы линейных уравнений.
  • Загрузка данных из CSV, статистический анализ (среднее, медиана, стандартное отклонение, перцентили), нормализация данных.
  • Визуализация: гистограмма, тепловая карта корреляции, линейный график.

Ход выполнения

Подготовка окружения

Было создано виртуальное окружение и установлены необходимые библиотеки: numpy, pandas, matplotlib, seaborn, pytest.

Реализация функций

Все функции написаны с использованием встроенных возможностей NumPy без явных циклов, что обеспечивает высокую производительность и читаемость кода. Код снабжён аннотациями типов (PEP-484), докстрингами (PEP-257) и соответствует стилю PEP-8.

Ключевые моменты реализации:
  • Для создания массивов использованы np.arange(10) и np.random.rand(5,5).
  • Изменение формы выполнено через метод .reshape().
  • Транспонирование – через свойство .T.
  • Векторные операции (+, * и np.dot) применены поэлементно и для скалярного произведения.
  • Матричные операции реализованы с помощью оператора @ и функций из np.linalg.
  • Загрузка данных выполнена через pandas.read_csv с последующим преобразованием в NumPy массив.
  • Статистические показатели рассчитаны функциями np.mean, np.median, np.std, np.percentile.
  • Для Min-Max нормализации использована формула (x - min) / (max - min).
  • Графики сохранены в папку plots/

Результаты

  • Все функции реализованы и протестированы.
  • Получены статистические характеристики данных (средний балл, разброс и т.д.).
  • Построены графики: гистограмма распределения оценок, тепловая карта корреляции между предметами, линейный график успеваемости студентов.

Выводы

В ходе работы были закреплены навыки использования NumPy для численных расчётов, векторных и матричных операций, а также визуализации данных с помощью Matplotlib и Seaborn. Полученный код соответствует стандартам оформления и может быть использован как основа для более сложных проектов по анализу данных.
